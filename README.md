# QuickVault Chrome Extension

QuickVault is a secure, local-first Chrome extension for managing secrets (like API keys) and quick notes (snippets) directly from your browser.

## Features
- **Secure Storage**: Secrets are encrypted with a PIN and stored locally.
- **Quick Paste**: Easily save and copy frequent text snippets.
- **Privacy First**: No data leaves your machine; everything is in `chrome.storage.local`.
- **Auto-Clear Clipboard**: Option to automatically clear copied secrets after a set time.
- **Dark Mode**: sleek, modern UI.

## Installation
1. Clone this repo.
2. Go to `chrome://extensions/` in Chrome.
3. Enable "Developer mode".
4. Click "Load unpacked" and select this directory.
